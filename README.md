# Site Web — Lingueo

Site vitrine B2B de Lingueo, orienté ETI et proposition de valeur commerciale.

## Structure

```
index.html     — Page principale (single-page scrollable)
```

## Sections

1. **Hero** — Proposition de valeur + stats clés + trust bar clients
2. **Pourquoi Lingueo** — 3 piliers différenciants + société + track record
3. **Nos produits** — FlashLevel / ENGAGE / LILATE
4. **Méthode Lingueo** — C.O.L. + service managé + KPIs
5. **Références clients** — 6 cas clients + verbatims
6. **Pour qui ?** — ETI / PME / Cabinets de recrutement
7. **Ressources** — Livres blancs + guide + outil ROI
8. **Contact** — Formulaire qualifiant

## Stack

- HTML / CSS / JS vanilla (aucune dépendance)
- Fonts : Bricolage Grotesque + Inter (Google Fonts)
- Responsive mobile/tablette/desktop

## Lancement local

```bash
open index.html
# ou
npx serve .
```
